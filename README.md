## Project Description
This project demonstrates interactive web pages using JavaScript DOM manipulation. 
You can practice selecting elements, traversing the DOM, modifying content, handling events, and building a functional To-Do List application. 
It also includes small daily challenges like color changing, dynamic paragraphs, and image removal.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

## Features
- DOM element selection (`getElementById`, `querySelector`, `querySelectorAll`)
- DOM traversal (parent, children, siblings)
- Modify text and HTML content dynamically
- Work with attributes and data attributes
- Add and remove elements dynamically
- Event handling for mouse, keyboard, form, and window events
- Event bubbling and delegation
- Interactive To-Do List:
  - Add, delete, complete tasks
  - Filter by all, active, or completed
  - Clear completed tasks
- Contact form validation
- Daily challenges:
  - Random heading color
  - Add paragraphs dynamically
  - Remove all images
